# New-LXMusic
New-LXMusic

原项目: https://github.com/lyswhut/lx-music-desktop

1.删除大量功能目前主要是搜索、歌单导入、新的壁纸功能

2.可以添加视频成为动态壁纸，应用最小化后动态壁纸会自动暂停减小GPU占用（原本的主题中设置背景图片功能可以不再使用了，用于更改主题颜色就行）

3.增加歌词翻译歌词可以左右拖动双击歌词区域自动归位

4.安装依赖，直接在根目录执行npm install，或运行脚本install-deps.bat等待安装完成后运行newlx-music.exe， Node.js >= 22


注：没有Node环境就在Releases中下载node_modules.rar，然后解压到根目录下
